Paramètres

p_diff_geom_in_The point by wich the ray enters the geometry
p_diff_geom_out_The point by wich the ray leaves the geometry
Renvoie
true if the ray intersects the geometry. In and out parameters are filled by this method. The default returned value is false such that non volumic geometries do not need to implement this.
