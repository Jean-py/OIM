getTexel Get a single texel from a texture.

This function access the texture at coordinates (row, column) and fetch the value of the corresponding texel (pixel from a texture)

Paramètres
pixelsThe image to access, organized as a linear array of texel arranged by row
widthWidth of the image
heightHeight of the image
depthDepth of the image (number of component by texel)
columnColumn coordinate of the requested texel
rowRow coordinate of the requested texel
Renvoie
The value of the texel
A faire:
Transfom the 2D coordinates in a 1D index and get the corresponding texel value

interpolateTexture Get a texel linearly interpolated from its neighbors

Paramètres
pixelsThe image to access, organized as a linear array of texel arranged by row
widthWidth of the image
heightHeight of the image
depthDepth of the image (number of component by texel)
sThe column coordinate of the requested texel as a floating point
tThe row coordinate of the requested texel as a floating point
Renvoie
The value of the interpolated texel
A faire:
From the floating point coordinates, compute the integer part and the fractional part. The integer part is the used to acces the 4 texels implied in the interpolation (One texel and its 3 up and right neighors), the fractional part is used to linearly interpolate from neighbors.

prefilterTexture Compute an array of images with geometrically decreasing resolution from the original image.

Paramètres
imagearrayThe array of images to compute. element at index 0 in this array is the full resolution image and must not be modified
widthWidth of the full resolution image
heightHeight of the full resolution image
depthDepth of the image (number of component by texel)
nblevelsNumber of level to compute : nblevels = log2(min(width, height))
Renvoie
if the array may be filled, return true, else return false
